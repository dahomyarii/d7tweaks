IF WIFI STILL DOESN'T APPEAR:
=============================

Try these steps in order:

STEP 1: Restart Your Laptop
---------------------------
Sometimes a full restart fixes connectivity issues.
  1. Click Start menu > Power > Restart
  2. Wait for laptop to fully restart
  3. Check if WiFi appears

STEP 2: Restart Your WiFi Router
---------------------------------
Your router might need a reset.
  1. Unplug power from WiFi router
  2. Wait 10 seconds (count slowly!)
  3. Plug power back in
  4. Wait 2-3 minutes for router to fully boot
  5. Try connecting to WiFi again

STEP 3: Check Device Manager for Driver Issues
-----------------------------------------------
  1. Right-click Windows Start menu
  2. Select "Device Manager"
  3. Find "Network adapters" section
  4. Look for your WiFi adapter (usually "Wireless" or "WiFi")
  5. Check if there's a yellow exclamation mark (!)
  6. If yes:
     a) Right-click the adapter
     b) Select "Update driver"
     c) Choose "Search automatically for drivers"
     d) Wait for update to complete
     e) Restart laptop

STEP 4: Check BIOS Settings
----------------------------
WiFi might be disabled in BIOS!
  1. Restart your laptop
  2. While booting, press F2, Del, or F10 (try each if unsure)
  3. Look for "WiFi", "WLAN", or "Integrated Peripherals"
  4. Make sure it says "ENABLED"
  5. Save and exit BIOS
  6. Laptop will restart
  7. Try WiFi again

STEP 5: Forget Network & Reconnect
-----------------------------------
The WiFi profile might be corrupted.
  1. Settings > WiFi > Manage Known Networks
  2. Find your network name
  3. Click on it > Forget
  4. Open WiFi list again
  5. Select your network
  6. Enter password again
  7. Wait for connection

STEP 6: Run Complete Rollback
-----------------------------
If nothing else works, completely restore Windows to original state.
  1. Right-click on: ROLLBACK.bat
  2. Select "Run as Administrator"
  3. Press any key to continue
  4. Wait 2-3 minutes for rollback to complete
  5. Choose "Y" to restart now
  6. Laptop will restart automatically
  7. After restart, WiFi should definitely work

IF YOU STILL HAVE PROBLEMS:
===========================

Check these things:

1. Is WiFi turned ON?
   - Look for a physical WiFi button on your laptop
   - Or press Fn + F2 (or Fn + WiFi key) to toggle WiFi
   - Check if WiFi icon in taskbar shows connected

2. Is router working?
   - Check if router lights are on and blinking
   - Can another device connect to this WiFi?
   - Try plugging an Ethernet cable if available

3. Is there a WiFi driver issue?
   - Check Event Viewer for network errors
   - Download latest WiFi driver from HP website
   - For AMD Ryzen: Get driver from AMD website

4. Is Windows Firewall blocking WiFi?
   - Settings > Privacy & Security > Windows Firewall
   - Make sure WiFi networks are not blocked

5. Do you need a clean Windows install?
   - If WiFi is still broken after all these steps
   - You may need to reinstall Windows
   - Contact HP support or professional technician
