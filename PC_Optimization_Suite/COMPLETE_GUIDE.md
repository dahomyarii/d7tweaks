## <a name="installation"></a>ðŸ”§ INSTALLATION GUIDE

### Requirements
- Windows 10 or Windows 11
- Administrator privileges
- 500 MB free disk space
- 30 minutes for full installation
- Backup of important data (recommended)

### Step-by-Step Installation

#### Step 1: Backup Your System
```
OPTIONAL but RECOMMENDED:
1. Create System Image
   - Settings > System > Recovery
   - Click "Create recovery drive"
   
2. Or use Windows Backup
   - Settings > System > Backup
   - Turn on "Backup now"
```

#### Step 2: Run Main Optimizer
```
1. Extract all files to a folder
2. Right-click RUN_ME_FIRST.bat
3. Select "Run as administrator"
4. If prompted, click "Yes" to allow
5. Choose Option 1: RUN ALL OPTIMIZATIONS
6. Wait for completion (10-15 minutes)
7. Choose "Yes" to restart when asked
```

#### Step 3: Verify Installation
```
After restart:
1. Open RUN_ME_FIRST.bat again
2. Choose Option 3: System Information
3. Verify changes were applied
4. Note your system specs
```

#### Step 4: Update GPU Drivers
```
CRITICAL STEP - Can give 5-15 extra FPS!

1. Run UPDATE_GPU_DRIVERS.bat
2. Follow on-screen instructions
3. Download latest drivers from:
   - NVIDIA: nvidia.com/Download
   - AMD: amd.com/drivers
   - Intel: ark.intel.com
4. Install driver with clean install option
5. Restart computer
```

#### Step 5: Apply Game Settings
```
1. Open GAME_SETTINGS.bat (optional, it shows settings)
2. Launch Call of Duty
3. Go to Settings > Graphics
4. Apply settings from GAME_SETTINGS_GUIDE.txt:
   - Fullscreen mode
   - Very Low graphics
   - Disable Ray Tracing
   - Disable DLSS/FSR
   - Short view distance
   - V-Sync: Off
```

#### Step 6: Test Gaming
```
1. Close ALL background applications
2. Launch your game
3. Play for 5-10 minutes
4. Check FPS (usually shows in corner)
5. Download HWiNFO64 to monitor temps
```

## <a name="troubleshooting"></a>ðŸ”§ TROUBLESHOOTING

### Problem: "Script requires administrator privileges"
**Solution:**
1. Right-click the .bat file
2. Select "Run as administrator"
3. Click "Yes" when prompted

### Problem: "Game FPS is worse than before"
**Solution:**
1. Update GPU drivers immediately
2. Check GPU temperature (may be thermal throttling)
3. Close all background applications
4. Run ROLLBACK.bat and try again
5. Apply scripts one at a time instead of all at once

### Problem: "Game won't launch"
**Solution:**
1. Run ROLLBACK.bat
2. Or use Windows System Restore:
   - Settings > System > Recovery
   - Click "Open System Restore"
   - Select "PC Optimization Suite Pre-Optimization"
3. Reinstall game
4. Try running scripts individually

### Problem: "System is very slow/laggy"
**Solution:**
1. Run ROLLBACK.bat
2. Wait for system to stabilize
3. Restart computer
4. Try again, but don't run all scripts at once
5. Check disk space (need at least 10% free)

### Problem: "Getting crashes in games"
**Solution:**
1. Check GPU memory is sufficient (in BIOS settings)
2. Lower game graphics settings further
3. Update GPU drivers to latest version
4. Check GPU temperature (under 85Â°C)
5. Check RAM usage (close other apps)
6. If crashes persist, run ROLLBACK.bat

### Problem: "High GPU/CPU temperatures"
**Solution:**
1. Clean laptop vents with compressed air
2. Buy external cooling pad ($20-40)
3. Lower graphics settings in game
4. Reduce resolution
5. Enable FPS cap in game (60 FPS)
6. Check ambient temperature (room temperature)
7. Use laptop on desk, not on bed/pillow

### Problem: "Can't find GPU in system information"
**Solution:**
1. Update chipset drivers first
2. Update BIOS (risky, check manufacturer)
3. Check Device Manager for unknown devices
4. Install Intel/AMD chipset drivers from OEM

### Problem: "Script errors or nothing happens"
**Solution:**
1. Make sure running as administrator
2. Close antivirus temporarily
3. Run one script at a time
4. Check if script file is corrupted
5. Re-download the files

### BIOS Tweaks (Advanced)
```
1. Increase DVMT Pre-Allocated Memory:
   - Restart computer
   - Press DEL or F2 during startup
   - Find: Integrated Graphics Settings
   - Set DVMT to 512MB or 1GB
   - Save and exit
   - Can improve FPS by 10-20%

2. Enable PCI-E Gen 3:
   - Same BIOS menu
   - Find: PCI-E Slot Speed
   - Set to Gen3
   - Save and exit

3. Disable Unused Devices:
   - Disable serial ports
   - Disable unused USB controllers
   - Disable floppy drive
   - Can save 5-10% performance
```

### Network Optimization (For Online Games)
```
1. Use Ethernet cable (wired)
   - Much faster than WiFi
   - Reduce latency by 50%
   - More stable connection

2. Close bandwidth hogs:
   - Disable cloud sync (OneDrive)
   - Close Dropbox/Google Drive
   - Close streaming apps
   - Close Windows Update
   - Disable Discord streaming

3. QoS Settings (if using home WiFi):
   - Log into router
   - Find QoS settings
   - Set gaming device to priority
   - Allocate more bandwidth
```
